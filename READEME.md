# Getting Started
## Requirements:
```
Spring Boot : 3.2.3
Java : 17
Maven 3.9 +
```
### Clone Github Repository
```bash
git clone https://github.com/deepaksorthiya/spring-boot-3-testing-web-complete
cd spring-boot-3-testing-web-complete
```
### Run Spring Boot App
```bash
mvn spring-boot:run
```
